# Projeto Financas ReactJS

## Sobre o projeto

E um projeto de controle de finanças pessoais, aonde o usuário vai poder adicionar novas entradas e novas saídas e no final vai ter um total.

## Técnologias utilizadas no projeto
- ReactJS
- Styled Components
- MirageJS
- Context API
- Hooks
- Axios
